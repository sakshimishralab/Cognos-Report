\# Data Analysis \& BI Reporting: Global Tech Sales Performance



\## Overview

This project utilizes \*\*IBM Cognos Analytics\*\* to transform raw operational data into a high-level Business Intelligence report. By implementing advanced data modeling and interactive visualizations, the project identifies regional profitability trends, product category performance, and operational anomalies to support data-driven executive decisions.



\## Objectives

\* \*\*Data Architecture:\*\* Developed a unified data module from raw CSV input to ensure consistent reporting across all visualizations.

\* \*\*Multi-Dimensional Analysis:\*\* Designed a comprehensive report layout featuring temporal trends, regional comparisons, and product-level distributions.

\* \*\*Interactive Visuals:\*\* Leveraged Cognos's reporting engine to create a synchronized view of tabular data and graphical insights.

\* \*\*Technical Mastery:\*\* Demonstrated proficiency in using \*\*Horizontal Bar Charts\*\*, \*\*Clustered Column Charts\*\*, \*\*Scatter Plots\*\*, and \*\*Detailed List Reports\*\*.



\## Dataset Profile

\* \*\*Name:\*\* Tech\_Sales\_Operations\_2026

\* \*\*Domain:\*\* Technology Retail \& Services

\* \*\*Attributes:\*\* Order ID, Transaction Date, Global Region, Product Category, Sales (USD), and Profit Margin (%).

\* \*\*Scope:\*\* Global transactions spanning from May 2025 to April 2026.



\## Visual Analysis \& Insights



\### 1. Temporal Profitability (Horizontal Bar Chart)

\* \*\*Focus:\*\* Analyzes Profit Margin % across specific dates.

\* \*\*Insight:\*\* Identified peak profitability periods in \*\*April 2026\*\* and \*\*November 2025\*\*, where margins consistently exceeded 45%, suggesting successful seasonal campaigns.



\### 2. Regional Performance (Clustered Column Chart)

\* \*\*Focus:\*\* Aggregated Profit metrics by Geographic Region.

\* \*\*Insight:\*\* \*\*North America\*\* represents the highest total profit volume, followed by \*\*Asia\*\*. This visualization highlights the scale of Western markets compared to emerging territories.



\### 3. Category Distribution (Scatter Plot)

\* \*\*Focus:\*\* Mapping Product names against their respective Categories (Hardware, Software, Services).

\* \*\*Insight:\*\* High density in the \*\*Hardware\*\* segment (e.g., Laptop Pro 15, Tablet Air) indicates a hardware-driven revenue model, while \*\*Services\*\* (e.g., AI Consultation) shows targeted, high-value placements.



\### 4. Operational Detail (List Report)

\* \*\*Focus:\*\* Granular view of every transaction including Return Status and Customer Segment.

\* \*\*Insight:\*\* Allows for immediate auditing of individual orders, such as \*\*ORD-1016\*\* (Large Software Sale) or identifying returns like \*\*ORD-1004\*\*.



\## Technical Implementation

\* \*\*Tool:\*\* IBM Cognos Analytics (Interactive Reporting Mode)

\* \*\*Modeling:\*\* Data items were categorized into Measures (Sales, Profit Margin) and Attributes (Region, Product) to allow for effective aggregation.

\* \*\*Layout:\*\* Used a grid-based dashboard layout to maintain visual hierarchy, ensuring the most critical charts are visible at a glance.



\## Conclusion

This project successfully demonstrates the transition from a flat dataset to a multi-layered BI report. Through IBM Cognos, I have created a scalable framework that provides both the "Big Picture" through visualizations and the "Fine Detail" through integrated list reports, fulfilling all requirements for the 2026 Mini Project.
